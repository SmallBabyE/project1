Вступление

На данном сайте вы узнаете как пользоваться языком HTML и как писать на нём интернет-страницы.

Начало работы на HTML

Для того чтобы начать работу на языке програмирования HTML, вы должны создать файл "index.html". После этого вы должны открыть его с помощью любого вам удобного браузера и открыть его с помощью блокнота или любого другого текстового редактора.

Первые строки кода

Код браузерной страницы похож на бутерброд, состоящий из определённых тегов, без которых сайт не заработает. Последовательность написания кода выглядит так: (html)(head)(/head)(body)(/body)(/html)

Значение тегов

Теги (head)(/head) используются для того чтобы задать параметры сайта, которые не отразятся на внешнем виде страницы. Тег (body)(/body) используется для изменения визуальной составляющей сайта, добавление текста, таблиц, отступов, линий - всё это изменяиется в теге (body)(/body).

Изменение тегов

Начнём наши изменения сайта с тегов (head)(/head). Между тегами "head" мы должны указать стандарт кодирования символов на нашей странице. Делается это с помощью самозакрывающегося тега (meta). (meta charset="UTF-8"). Теперь изменим название для нашей страницы, которое будет отображаться в заголовке, сделаем это с помощью тегов (title)(/title). Для начального этапа нашего обучения нам достаточно этих двух тегов в разделе head. Поэтому мы можем приступить к изменению раздела body. Всё, что написано выше на странице, оно написано в рамках тегов body. Для написания заголовка мы используем тег (h)(/h), добавляя к нему цифру от 1 до 6, чем больше цифра, тем меньше размер заголовка. Для создания параграфов используется тег (p)(/p). Для того чтобы добавить линию между нашими заголовками и параграфами мы можем воспользоваться самозакрывающимся тегом (hr) Давайте для примера напишем простой заголовок, текст и отделим их линиями от предидущих частей моего обучения. (hr)(h2)Заголовок(/h2)(p)Текст(/p)(hr)

Начало работы на CSS

Для начала нам нужно создать файл с любым названием и расшерением .css. В моём случае он будет называться "styles.css". Для слияния нашего css-файла с нашим сайтом, мы должны использовать самозакрывающийся тег в рамках тегов head (link href="Название вашего файла.css rel="stylesheet").

Цвет фона страницы

Для того чтобы задать цвет фона интернет-страницы, нам нужно открыть наш css файл в любом текстовом редакторе и обратиться к тегам body. body {backgrond: цвет(в нашем случае будет бежевый (beige))}

Изменение ширины страницы.

Для изменения ширины страницы нам нужно обернуть в div весь наш сайт(я оберну только всё то что идет после цвета фона страницы). (div class="название класса(на английском языке)"всё что вы напишете в тегах body(/div). Обращаемся через точку к классу с нужным названием и задаём ему параметры ширины страницы. (.название класса {max-width: 750px(данный параметр отвечает за ширину страницы в пикселях); margin: auto;})

Создание визуальной составляющей объекта

Чтобы задать цвет объекта, нам нужно обернуть его в div и изменить цвет с помощью команды background-color : (ваш цвет). Чтобы изменить границы объекта нам нужно использовать команду padding: кол-во пикселей сверху и снизу(px), кол-во пикселей (px) по бокам.

Улучшение визуализации объектов

Цвета и отступов недостаточно, поэтому, я предлагаю добвить нашим объектам красивые края, тени, отступы. Изображения

Для того, чтобы добавить на нашу страницу картинку, нам потребуется самозакрывающийся тег img. (img src="название изображение.расшерение" class="название класса к которому мы будем обращаться, когда будем стилизовать наше изображение")

Стилизация изображения

Для наглядности я использовал очень большую картинку и выглядит это плохо, но это можно исправить, изменив размер, добавив ей тени, отступы и красивые границы. Для изменения размера картинки, используем команду width: (кол-во пикселей(px)); Для добавления теней испоользуем уже обговорённый ранее - box-shadow, для отсупов - margin-bottom, для красивых границ - border-radius.

Добавление простых приложений на страницу

Для того чтобы добавить простое приложение на страницу, нам нужно воспользоваться тегом (iframe)(/iframe). Для примера, давайте используем календарь. Так как написать полностью рабочий календарь, используя язык HTML невозможно, мы воспользуемся общедоступным яндекс-калндарём. Ссылка на него есть в общем доступе. (iframe src="ссылка" class="название класса")(/iframe). Далее обратимся к нашему календарю через css-файл и зададим ему стилистику через ранее изученные команды. Я изменил края календаря (border-radius), ширину (width), высоту (height) и добавил тень (box-shadow).
